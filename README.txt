Mario Kart Tour SoundExtractor


To use this tool, Android must be rooted and able to bypass SafetyNet.

1. Copy the files in "/data/data/com.nintendo.zaka/files/Nabe" on Android to the "input_nabe" folder.
2. Run bnk_to_wem.py or bnk_to_wem.exe.
3. Wait for it to finish.
4. The audio file is output to the output_nabe folder. enjoy!
