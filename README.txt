Mario Kart Tour SoundExtractor


To use this tool, your Android needs to be rooted and able to bypass SafetyNet.

1. Copy the contents of "/data/data/com.nintendo.zaka/files/Nabe" on Android to the "input_nabe" folder.
2. Run bnk_to_wem.py or bnk_to_wem.exe.
3. Wait for it to finish.
4. The audio file will be output to the "output_nabe" folder. 
5. Enjoy!
